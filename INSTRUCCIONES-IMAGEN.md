# Instrucciones para la imagen de fondo de partitura

Para completar el diseño, necesitas agregar una imagen de fondo de partitura musical.

## Opción 1: Guardar una imagen existente
Si tienes una imagen de partitura musical, guárdala como `sheet-music-bg.png` en la carpeta del proyecto:
```
C:\Users\HP SUPPORT\.gemini\antigravity\scratch\casa-del-pianista\sheet-music-bg.png
```

## Opción 2: Descargar una imagen gratuita
Busca en sitios como:
- Unsplash.com (busca "sheet music")
- Pexels.com (busca "piano sheet music")
- Pixabay.com (busca "musical notes")

Descarga una imagen de alta calidad y guárdala como `sheet-music-bg.png`.

## Características recomendadas de la imagen:
- Alta resolución (mínimo 1920x1080px)
- Fondo claro o blanco
- Notas musicales claramente visibles
- Estilo elegante y profesional

El CSS ya está configurado para aplicar automáticamente una capa blanca semitransparente (85% de opacidad) sobre la imagen, lo que hará que se vea sutil y elegante detrás del contenido.
